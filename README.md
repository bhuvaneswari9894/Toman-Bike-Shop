 Bike Shop Sales Analysis (SQL + Power BI)

📌 Project Overview

This project analyzes bike shop sales data using SQL and Power BI to generate actionable business insights. SQL is used for data extraction, cleaning, and transformation, while Power BI is used to build interactive dashboards that visualize revenue, profit, seasonal trends, and rider demographics.

🎯 Objective

To monitor sales performance and identify key trends that help improve business decision-making.

🛠 Tools & Technologies

* SQL
* Power BI
* Data Visualization
* Business Intelligence
* 
 📊 Key Analysis

* Revenue and profit analysis
* KPI tracking over time
* Seasonal revenue trends
* Rider demographic analysis
* Peak sales hours identification

#📈 Outcome

The dashboard reveals profitable time periods, customer distribution patterns, and overall sales performance, enabling data-driven strategies for business growth.
